📖 Инструкции

Добавление скриптов

  1. Создайте файл скрипта с расширением .sh, .bash или .py
  2. Поместите файл в папку scripts/
  3. Добавьте описание в первую строку комментария: # Описание скрипта
  4. Для shell-скриптов сделайте файл исполняемым: chmod +x script.sh

Примеры

Bash скрипт (example.sh): 
#!/bin/bash
# Пример скрипта - выводит информацию о системе

echo "=== Информация о системе ==="
uname -a
echo ""
echo "=== Свободное место ==="
df -h
echo ""
echo "=== Память ==="
free -m
Python скрипт (example.py): 
#!/usr/bin/env python3
# Пример Python скрипта

import sys
import platform

print("Python версия:", sys.version)
print("Платформа:", platform.platform())

Ограничения

  • Максимальное время выполнения: 5 минут
  • Скрипты выполняются от имени пользователя приложения
  • Для команд с sudo требуется настройка sudoers
